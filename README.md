# AI for Education

This project implements a Retrieval-Augmented Generation (RAG) system with **two agents** designed for math tutoring for **middle school students**.

## Main idea

- The system uses **2 RAG agents**
- **Agent 1 (Student agent)**: acts as a student and generates an incorrect reasoning for a math problem.
- **Agent 2 (Evaluator agent)**: evaluates the reasoning, detects mistakes, and instructs Agent 1 on the next step of the interaction.

The goal is to simulate a learning process where the student critiques, debugs, and explains why the AI’s reasoning is incorrect .
